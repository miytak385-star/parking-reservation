# 駐車スペース予約アプリ

マンションの駐車スペース予約・承認フローをWeb化したアプリです。

---

## なぜ作ったか

マンション住民が駐車スペースを利用するには、管理人への口頭・電話問い合わせが必要でした。  
土日祝日は管理人が不在のため予約・確認ができず、紙の予約一覧は共用スペースまで行かないと見られない状態でした。

このアプリで**予約・承認フローをWeb化**し、時間・場所を問わず予約・確認できる環境を整えることを目的としています。

## 機能概要

### 住民向け
- Google / メール＋パスワードでログイン・新規登録
- カレンダーで空き状況を確認（日付セルを色分け表示）
- SVG簡易地図でスペースを視覚的に選択
- 予約申請・申請完了通知
- マイ予約の確認・キャンセル

### 管理人向け
- 全予約の一覧表示・並び替え・フィルタリング
- 予約の承認・否認（住民へメール通知）
