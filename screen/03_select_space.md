```
┌─────────────────────────────────────────┐
│  🅿 駐車スペース予約         [マイ予約] │
├─────────────────────────────────────────┤
│                                         │
│  2026年5月25日 のスペースを選択         │
│                                         │
│  ┌───────────────────────────────────┐  │
│  │        マンション                 │  │
│  │                                   │  │
│  │  [住民駐車場] |壁|  [P1] [P2]  [P3]│  │
│  │                                   │  │
│  └───────────────────────────────────┘  │
│                                         │
│  ※ P3は斜め配置（軽自動車専用）        │
│                                         │
└─────────────────────────────────────────┘
```

## 色の定義

| 色 | 状態 |
|---|---|
| 青 | 空き（クリック可） |
| 灰 | 予約済み（クリック不可） |

## 動作
- SVGで駐車場レイアウトを描画（`docs/地図イメージ.png` 参照）
- P1・P2は縦向き、P3は斜め配置
- 住民駐車場・壁・マンションはクリック不可の背景要素
- スペースをクリックすると予約申請フォームへ遷移
