# カレンダー画面 `/`（ホーム）

```
┌─────────────────────────────────────────┐
│  🅿 駐車スペース予約         [マイ予約] │
├─────────────────────────────────────────┤
│          < 2026年5月 >                  │
├────┬────┬────┬────┬────┬────┬────┤
│ 日 │ 月 │ 火 │ 水 │ 木 │ 金 │ 土 │
├────┼────┼────┼────┼────┼────┼────┤
│    │    │    │    │    │  1 │  2 │
│    │    │    │    │    │ 緑  │ 緑  │
├────┼────┼────┼────┼────┼────┼────┤
│  3 │  4 │  5 │  6 │  7 │  8 │  9 │
│ 緑  │ 緑  │ 黄  │ 灰  │ 緑  │ 緑  │ 緑  │
├────┼────┼────┼────┼────┼────┼────┤
│ 10 │ 11 │ 12 │ .. │ .. │ .. │ .. │
│ 緑  │ 緑  │ 灰  │ .. │ .. │ .. │ .. │
└────┴────┴────┴────┴────┴────┴────┘
```

## 色の定義

| 色 | 状態 |
|---|---|
| 緑 | 全時間帯で空きあり |
| 黄 | 一部時間帯のみ空き |
| 灰 | 終日満車 |
| グレーアウト | 過去日（選択不可） |

## 動作
- MUIの `DateCalendar` を使用し、`day` スロットで日付セルをカスタマイズ
- 日付ホバー時にスペースごとの空き状況をツールチップで表示（例：P1 空き／P2 満車／P3 空き）
- 日付をクリックするとスペース選択画面へ遷移
